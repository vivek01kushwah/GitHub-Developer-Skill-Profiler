## 🚀 Quick Start

### For Windows Users:
```bash
# 1. Run the setup script (one time only)
setup.bat

# 2. In one terminal, start the backend:
run_backend.bat

# 3. In another terminal, start the frontend:
run_frontend.bat

# 4. Open http://localhost:3000 in your browser
```

### For Linux/macOS Users:
```bash
# 1. Run the setup script (one time only)
chmod +x setup.sh
./setup.sh

# 2. In one terminal, start the backend:
cd backend && source venv/bin/activate && python app.py

# 3. In another terminal, start the frontend:
cd frontend && npm start

# 4. Open http://localhost:3000 in your browser
```

## 🛠 Troubleshooting

### Issue: Python not found
**Solution:** Install Python 3.9+ and add to PATH

### Issue: Node.js not found  
**Solution:** Install Node.js 16+ and add to PATH

### Issue: CORS errors
**Solution:** Ensure backend is running at http://localhost:5000

### Issue: GitHub API rate limit
**Solution:** Add GitHub Personal Access Token to backend/.env

### For all issues:
1. Run `verify.bat` to check setup
2. Check SETUP.md troubleshooting section
3. Ensure both servers are running
